# 末世重生章节规划

## 章节节拍基本结构

```
开头（200字内）→ 承接生存危机或悬念
中段（1800-2000字）→ 3-5个清晰节拍
结尾（200-300字）→ 新威胁/爽点/情感触动
```

---

## 节拍类型库（每章3-5个）

| 节拍类型 | 触发条件 | 核心效果 |
|---------|---------|---------|
| **生存压力** | 物资不足/丧尸群迫近/据点受威胁 | 持续紧张感，避免故事松弛 |
| **预知展示** | 主角"恰好"知道正确路线/判断 | 爽感，同时引发队友疑惑 |
| **异能使用** | 战斗/侦查/物资搬运 | 展示成长，空间异能多场景活用 |
| **人性冲突** | 资源分配/逃跑vs留守/信任危机 | 灰度张力，非黑即白最无聊 |
| **打脸预谋** | 某人的质疑/嘲讽→主角隐忍→事后证明 | 节奏高潮 |
| **情感推进** | 共患难/误解化解/守护行为 | 情感线推进 |
| **预知偏差** | 某件事和上世不一样了 | 制造悬念和不安 |
| **据点发展** | 新成员加入/物资建设/防御加固 | 满足感，势力成长 |
| **异能突破** | 战斗极限/晶核吸收 | 爽点，风险并存 |
| **上世仇人** | 仇人出现/过去记忆闪回 | 情感张力，布局伏笔 |

---

## 末世节奏控制原则

- **生存威胁不能消失超过3章**：读者随时需要感受到末世的危险
- **连续物资章节上限2章**：不能连续多章只写"搜索物资"，要嵌入冲突
- **预知使用频率**：前期每章可用，中期每3-5章一次，后期稀缺化

### 末世特有节奏公式：

```
危机迫近（读者紧张）→ 主角预知应对（读者松了一口气）
           ↓
→ 预知出现偏差（读者又紧张了）→ 主角临场判断（悬念）→ 解决（爽/悲）
```

## 结尾钩子类型（末世版，必须具体）

| 类型 | 示例（具体画面/台词，≤50字）|
|------|--------------------------|
| 危机钩子 | 警报刺耳地响了。侦察员跑进来，脸色发白："东边来了，三千只以上。" |
| 爽点钩子 | 陈磊从空间里拿出第三箱子弹，扔到桌上。"够用三个月了。" |
| 预知失效钩子 | 这里应该是安全的。他确定。可是门缝里，有脚印。新的。 |
| 人性钩子 | 李刚没有转身。只说了一句话："我先走。" |
| 情感钩子 | 她停下脚步，没有回头，声音压得很低："谢谢你救我。" |
