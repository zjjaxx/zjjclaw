# 末世异能体系设计

## 体系设计原则

末世异能体系与修真体系最大的不同：
- **修真**：通过修炼慢慢积累，主动成长
- **末世**：靠杀丧尸+晶核进化，每次进化都伴随危险

末世体系必须体现**危险与收益并存**的紧张感。

---

## 标准异能等级体系

| 等级 | 称谓 | 战力描述 | 人口占比 |
|------|------|---------|---------|
| 一阶 | 觉醒者 | 刚觉醒，勉强比普通人强 | 约0.8% |
| 二阶 | 初阶能力者 | 可以单独清理小群丧尸 | 约0.15% |
| 三阶 | 中阶能力者 | 一队之核心战力 | 约0.04% |
| 四阶 | 高阶能力者 | 据点级别的战略资源 | 极稀少 |
| 五阶 | 超级能力者 | 区域性战略力量 | 凤毛麟角 |
| 六阶以上 | 超凡者 | 可改变末世格局的存在 | 全球个位数 |

## 异能类型与战场表现

| 异能系 | 核心优势 | 局限性 |
|-------|---------|-------|
| **火系** | 范围伤害强，清理丧尸效率高 | 高耗能，不适合精细控制 |
| **冰系** | 控制能力，可封冻 | 需要持续输出维持 |
| **雷系** | 速度/破甲，高爆发 | 释放不稳定，伤及队友风险 |
| **空间系** | 位移/储物/切割（稀有高端用法）| 空间系极稀缺，储物是最常见初级应用 |
| **植物系** | 持续生产食物，防御陷阱 | 无进攻爆发力 |
| **心灵系** | 侦查/干扰/控制人心 | 近战脆弱，高风险暴露 |
| **治愈系** | 最受追捧的异能，各势力争抢 | 无攻击能力，永远是被保护对象 |
| **力量增幅系** | 肉搏无敌，抗打 | 速度受限，高阶丧尸有速度优势 |
| **速度系** | 机动，侦查，补刀 | 单次攻击力弱 |

## 主角金手指（末世重生版）

**组合金手指设计：重生记忆 + 空间异能**

| 金手指组件 | 具体能力 | 使用限制 |
|-----------|---------|---------|
| **重生记忆** | 知道末日爆发时间/关键物资地点/高阶丧尸出现规律/势力走向 | 记忆不完整（越到后期历史偏差越大） |
| **空间异能（储物空间）**| 任意存取物品，不受体积限制 | 空间容量需随等级扩大/初期容量有限 |
| **提前觉醒优势** | 末日前已知如何觉醒，第一批获得异能 | 异能等级初始仍是一阶，需正常进化 |

## 晶核系统（进化核心机制）

- **获取**：击杀丧尸后从丧尸颅内取出
- **用法**：同系晶核→直接进化；异系晶核→辅助训练但效率低
- **风险**：高阶晶核强行吸收会导致异能暴走/变异失败（死亡）
- **市场价值**：晶核是末世硬通货，可交换物资/服务

晶核品质对照表：
| 晶核品质 | 来源 | 对应进化 |
|---------|------|---------|
| 灰色（劣）| 普通丧尸 | 0→1阶，效率低 |
| 白色（普通）| 1-2阶丧尸 | 1→2阶 |
| 绿色（良）| 2-3阶丧尸 | 2→3阶 |
| 蓝色（优）| 3-4阶丧尸 | 3→4阶 |
| 紫色（稀有）| 4阶以上王者 | 大幅提升 |
