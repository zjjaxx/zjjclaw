# 玄幻修真章节审校

## 审校核心维度

### 1. 世界观一致性
- 境界描写是否前后一致（不能金丹境界打不过筑基）
- 法宝/功法名称是否与已有设定匹配
- 地点描写是否符合世界观设定

### 2. 修真气质保持
- 灵气描写是否有物理感（非漂浮式）
- 法术/斗法是否有具体动作，非"他出手了"
- 高境界修士的气场是否体现了压迫感

### 3. 节奏与爽感
- 修炼场景是否控制在章节20%以内
- 打脸场景是否存在拖节奏问题（铺垫过长）
- 章节结尾是否有具体的悬念钩子

### 4. 人物声音
- 各人物对话方式是否有差异化
- 主角是否过于啰嗦（话多减分）
- 反派是否太蠢（需有自己的逻辑）

### 5. 情节密度
- 章节是否推进了：修炼/打脸/情感/伏笔 至少一项
- 是否存在整章都是"过渡"的情况

---

## 输出格式

给出：
1. **整体评分**（满分10分）
2. **优点**（2-3条，具体到段落/场景）
3. **问题清单**（每条问题含：问题描述 + 修改建议）
4. **关键修改建议**（最需要改的1条，附改写示例）
