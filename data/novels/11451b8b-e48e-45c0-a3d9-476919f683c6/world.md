# 《重生末日》世界观设定

## 末日爆发时间线（华夏·K病毒纪元）

*   **Day 0 （天灾日）**：一场全球性的血色流星雨过后，未知的“K病毒”（Kaleidoscope Virus，因其引发千奇百怪的变异而得名）通过空气与水循环在极短时间内全球传播。约30%的人口在24小时内转化为只知啃食的“活尸”（初始丧尸）。通讯网络因太阳风暴干扰及人为切断（防止恐慌）而大面积中断。
*   **Day 1-7 （大崩坏周）**：城市秩序彻底崩溃。交通瘫痪，幸存者被困。初期活尸行动缓慢但数量恐怖，人口密集的都市区沦为地狱。地方政府与基层武装力量（派出所、武警支队）尝试建立临时庇护所，但因信息不通、资源匮乏及内部感染，多数在数日内陷落。广播中最后的官方声音是要求幸存者“就地坚守，等待救援”。
*   **Day 8-30 （撤离与绝望之月）**：大规模的人口死亡潮与迁徙潮。部分军队成功集结，依托军事基地、大型粮仓或地势险要之处建立“官方避难区”，但容量有限，且内部权力与资源斗争激烈。大量幸存者开始自发向郊区、山区、乡村流动，形成最初的流浪团伙和微型聚落。变异动物开始零星出现。
*   **Day 31-180 （势力分化期）**：K病毒催生的变异进入新阶段。丧尸中出现首批进化体（一阶），力量、速度或防御显著增强，对普通幸存者构成压倒性威胁。与此同时，人类中的异能觉醒者开始崭露头角。失去联系的各大避难区、民间据点逐渐稳固，但因理念、资源和领袖性格差异，分化成不同性质的势力。远程无线电通讯在小范围内恢复，末世“广播台”出现，交换信息与警告。
*   **Day 181+ （新秩序黎明）**：故事开始时点。全球人口估计已骤降至末日前5%-10%。稳定的、以物易物和武力为后盾的末世秩序在区域范围内初步形成。国家概念已然模糊，以“基地”、“聚集地”、“庇护所”为单位的割据局面成为常态。对丧尸晶核（能量结晶）的认知与应用开始普及，它成为了新的硬通货和力量源泉。

## 丧尸进化体系

丧尸的进化源于体内K病毒与宿主细胞、环境相互作用的深度结合，其大脑或心脏处会凝结出颜色各异的“晶核”，是其能量核心与弱点。

| 阶段 | 名称（民间俗称） | 特征与能力 | 晶核与威胁等级 |
| :--- | :--- | :--- | :--- |
| **0阶** | **活尸 / 行尸** | 行动缓慢，关节僵硬，依靠听觉与嗅觉（对血腥味极度敏感），无智力，基础感染源。 | **白色浑浊晶核**。能量低下，主要威胁在于无休止的数量与尸潮。 |
| **1阶** | **疾行者 / 厚甲尸** | 出现初步分化：或速度显著提升（疾行者），或皮肤角质化、肌肉硬化（厚甲尸）。开始有微弱的环境适应本能。 | **淡绿色（速度系） / 淡黄色（防御系）晶核**。需要训练有素的普通人小队或低阶异能者才能稳妥应对。 |
| **2-3阶** | **特化体 / 猎杀者** | 能力进一步特化并可能组合。如喷射腐蚀性毒液、长出骨刃、具备短暂攀爬或跳跃能力（猎杀者）。出现简单的捕猎协作。 | **亮绿色/亮黄色/淡蓝色（元素系）晶核**。单体即可对中小型幸存者据点造成严重破坏，通常需要同阶或更高阶异能者剿灭。 |
| **4-5阶** | **统领者 / 兽化尸王** | 初步恢复部分生前记忆碎片或诞生低级智能，能通过精神波动或嘶吼指挥低阶丧尸形成有战术意图的尸群。体型可能巨大化或异化成非人形态。 | **深蓝/紫色晶核**。区域级灾难，能轻易摧毁一个缺乏重型火力的中型据点。其出现往往意味着一个“尸巢”正在形成。 |
| **6阶+** | **变异领主 / 天灾形态** | 高智能，保有部分人类思维模式或形成全新的残酷意志。能建立并经营“尸巢”，将一片区域转化为高浓度病毒污染区，培育低阶丧尸，甚至有驱使变异兽的迹象。能力开始涉及精神、元素操控等高级领域。 | **金色/暗金色/多彩晶核**。等同于移动的天灾，是各大人类势力最高级别的警戒与讨伐目标，其晶核是引发势力战争的导火索。 |

## 异能觉醒机制（华夏特色）

*   **觉醒比例与触发**：约1.2%的华夏幸存者能在极端压力、生死边缘或主动吸收纯净晶核能量后觉醒。部分古武修习者、精神意志极端坚定者或有特殊天赋（如天生念动力敏感）的人觉醒概率更高。
*   **异能类型**：
    *   **元素系**：金、木、水、火、土、雷、冰、风等。最常见，成长曲线明确。
    *   **强化系**：力量、速度、体能、感官、骨骼硬化、肌肉膨胀等。通常与古武术法结合能发挥更强战力。
    *   **特异系**：**空间**（稀有，初期多为存储，高阶涉及切割、禁锢）、**心灵**（精神感知、暗示、操控，极度稀有）、**治愈**（战略级资源）、**召唤/契约**（与变异兽或特殊存在建立联系）。
    *   **功法传承系**（独特设定）：极少数觉醒者能触发血脉或精神中的古老印记，获得残缺的“上古功法”记忆，其异能运行方式更接近修炼，潜力巨大但信息不全，需要自行摸索或寻找遗迹补全。
*   **等级划分**：一至九阶，每三阶为一个大门槛（低阶、中阶、高阶）。**三阶**是一个重要分水岭，异能将发生质变（如火焰从喷射变为塑形）；**六阶**可称“超凡”，能力开始影响规则雏形；**九阶**只存在于传说。
*   **进化方式**：1) **吸收同系或无属性晶核**（主渠道）；2) **极限战斗与生死磨练**；3) **修炼功法或观想图**（稀少）；4) **服用天材地宝**（变异植物/矿物精华）。

## 势力格局（故事开始时，主角重生时间点）

1.  **北方“钢铁长城”联合指挥部**：由数个大军区残部整合而成，占据旧工业基地和部分粮仓，组织度最高，保有相对完整的重工业能力和纪律部队。理念偏向保守与集权，致力于恢复旧秩序，对异能者采取“登记、征召、管制”政策。
2.  **江南“自由商会”联盟**：以数个大型民间幸存者基地为核心，依托长江水道形成的贸易同盟。商业氛围浓厚，流通相对自由，异能者地位很高，但内部竞争激烈，奉行弱肉强食的丛林法则。消息最灵通，黑市繁荣。
3.  **西南“山城庇护所”及周边苗彝自治寨**：利用复杂山地地形建立的分散式据点网络。民间自治色彩浓厚，部分寨子保留了独特的巫蛊传承，能与变异动植物达成微妙共存，对外界警惕性高。
4.  **沿海“救世军”与掠夺者团伙**：在秩序崩溃的沿海废墟地带，既有以极端宗教或理想主义凝聚的“救世军”，也有纯粹以掠夺为生的海盗式暴徒团伙。环境最混乱，机遇与风险并存。
5.  **散布各地的独立据点与流浪车队**：数量最多的生存形态。可能是一个加固的村庄、一个工厂、一个监狱或一支在公路上迁徙的车队。它们可能依附于大势力，也可能在夹缝中艰难求生。

## 主角所在据点（故事起点）：“望山镇”前哨营地

*   **位置与构成**：位于华东某省，依托一个末世前正在开发、人口相对稀少的山区旅游小镇“望山镇”建立。背靠山林，前有旧省道，地势较高，易守难攻。营地主要由三部分组成：1) 小镇原部分建筑加固而成的居住区；2) 镇外废弃小学改造的种植区和畜栏（尝试中）；3) 山顶观景台改造的瞭望哨。
*   **人口与领导**：约350名幸存者。领导层为“三人议事会”：前镇派出所副所长**老陈**（威望者，代表普通民众），土系一阶巅峰异能者**石勇**（主要战力代表），以及原镇卫生院唯一幸存的医生**苏婉**（技术与人道代表）。决策需至少两人同意。
*   **生存状况**：**勉强维持**。食物主要靠轮流外出搜寻附近废弃村庄，库存长期处于警戒线。水源依靠山泉和收集雨水，有简单的过滤装置。药品极度匮乏。防御主要依靠地形、简易路障、弓箭和少量自制武器，拥有三把私藏手枪和不到五十发子弹（由老陈严格管控）。营地内气氛紧张但尚存基本秩序，偷窃食物是唯一死罪。
*   **外部环境**：
    *   **威胁**：镇外省道上游荡着数百规模的零星尸群，偶尔有**一阶疾行者**混入其中。山林中有变异野猪和毒虫的传闻。三十公里外有一个名为“狼牙”的掠夺者团伙在活动，曾派来使者要求“上供”。
    *   **机遇**：七十公里外是传闻中规模较大的“江北幸存者基地”。东南方向五十公里处，有一个末世前的**区域物流中转中心**，已知有大量库存，但被一个疑似有**二阶厚甲尸**统领的尸群占据，周边势力均未敢轻易触动。
*   **主角初始位置**：主角重生归来，身份是营地中一个沉默寡言、刚刚（在他人看来）侥幸觉醒**微弱空间异能**（仅能储物约一立方米，对外宣称）的普通青年。他熟知未来几个月此地将在尸潮和“狼牙”团伙的双重打击下覆灭，也清楚那个物流中心的秘密与危险。他的空间里，已悄然囤积了始于末日爆发前、远超旁人想象的生存物资与关键工具。他的重生，将从改变这个小小营地的命运开始。
