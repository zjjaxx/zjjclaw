# 都市异能力量体系设计

## 设计原则

好的网文力量体系需要满足：
1. **层次分明**：境界划分清晰，每层都有明显差距
2. **可量化感**：让读者直观感受到强弱对比
3. **爽感设计**：主角突破时要有明显的质变
4. **开放上限**：为主角成长留足空间

## 境界设计模板

### 标准七阶结构（推荐）

```
凡人（普通人）
   ↓ 觉醒/踏入修炼门槛
一阶：[名称]（城市中的小喽啰）
二阶：[名称]（有一定地位）
三阶：[名称]（家族高层/帮派头目）
四阶：[名称]（城市级强者）
五阶：[名称]（省级/区域强者）
六阶：[名称]（国家级/顶尖强者）
七阶+：[名称]（传说中的存在）
```

每阶细分 3 个小境界（初期/中期/后期 或 一层/二层/三层）

### 必须说明的内容

1. **晋升条件**：如何从一阶晋升二阶（资源/顿悟/试炼）
2. **战力参考**：每阶能做到什么（一阶：轻松打倒10个普通人；三阶：能撕裂钢铁）
3. **主角金手指的特殊性**：为什么主角能快速突破
4. **资源体系**：修炼需要什么（灵石/经验值/异兽精血/秘境机缘）

## 与金手指的联动

- **系统流**：等级/属性点/技能点与境界对应
- **传承流**：传承功法对应哪些境界的快速突破
- **神器流**：神器提供的特殊加成如何体现

## 特殊能力种类（选3-5种填充世界）

都市异能常见能力分类：
- 元素系（火/水/雷/冰/风/土）
- 体质系（力量/速度/防御）
- 精神系（幻觉/读心/预知）
- 空间系（瞬移/储物/结界）
- 特殊系（治愈/诅咒/时间）

## 输出格式

清晰的 Markdown 表格 + 文字说明，包含：
- 境界总览表
- 各阶战力说明
- 晋升方式
- 主角金手指加成规则
- 世界中常见的能力种类
