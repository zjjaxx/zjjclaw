# 都市异能世界观构建

## 核心原则

都市异能世界观必须满足：**现代都市表面 + 隐藏的异能/修炼体系共存**。

## 必须构建的世界观要素

### 1. 城市背景
- 虚构但真实感强的城市名（华都、天南、玄城等）
- 城市规模与地位（省会/直辖市/特殊经济区）
- 城市特色（金融中心/军事重镇/古城）

### 2. 异能/修炼的存在方式
- 异能者在普通社会中如何隐藏身份
- 觉醒比例（万里挑一 / 百里挑一）
- 政府/军队是否知晓异能者存在
- 是否有专门管理异能者的机构

### 3. 势力格局（从小到大）
```
个人 → 家族/帮派 → 门派/组织 → 城市级势力 → 全国势力 → 更高层
```
描述主角所在城市的主要势力：
- 2-3 个家族（主角家族必须有落魄背景）
- 1-2 个秘密组织（异能者协会/修炼者联盟）
- 军方暗线（后期可拉拢的强援）

### 4. 普通人与异能者的边界
- 异能者是否公开存在
- 法律对异能者有无特殊规定
- 异能者的社会地位

### 5. 更大世界的暗示
- 隐藏在都市背后的更深秘密（上古遗迹/神秘空间/域外势力）
- 给主角后期成长留下足够空间

## 输出格式

用 Markdown 结构化输出，包含以上所有要素，语言生动具体，给人真实感。
字数约 800-1200 字。
