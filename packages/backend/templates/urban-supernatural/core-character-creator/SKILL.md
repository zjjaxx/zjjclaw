# 都市异能核心人物创建

## 核心人物定义

核心人物 = 贯穿全书的主线角色，从第一弧就开始埋线，最后一弧仍然有戏份。

**只创建以下角色，不多不少：**

| 类型 | 数量 | 说明 |
|------|------|------|
| protagonist | 1 | 主角 |
| female_lead | 1-2 | 主要女主（贯穿全书的感情线） |
| villain_boss | 1 | 最终大Boss（前期不出场，但要埋线） |
| supporting | 2-3 | 铁哥们/师父/家族长辈（全书陪伴型） |

**总计 5-7 个角色**，不要创建弧线特有的炮灰和中级反派。

## 主角设计要求

- **外表**：普通甚至寒酸（反差爽感）
- **性格**：表面淡定/内心热血，关键时刻霸气
- **初始境遇**：必须有"低谷"——被嘲笑、被退婚、被欺负
- **核心驱动**：保护家人/复仇/证明自己
- **禁忌**：不能太圣母，不能太傻白甜

## 女主设计要求

核心女主必须是全书感情线的主轴，不是某一弧的过客：
- 要有独立人格和自己的成长线
- 与主角的关系要有"不可替代性"（青梅竹马/命运牵绊/救命之恩）
- 性格鲜明，不能是花瓶

## 大Boss设计要求

- 与主角有**根本性矛盾**（杀父之仇/夺宝之恨/大道之争）
- 前期只通过线索暗示其存在
- 实力远超主角当前境界
- 必须有令人信服的动机，不能纯粹为坏而坏

## 核心配角设计要求

- **铁哥们/搭档**：全书陪伴，有自己的成长线，不只是工具人
- **师父/引路人**：识别主角潜力，给予第一次机缘，但不会长期无限罩着主角
- **家族长辈**：提供情感驱动（需要主角保护的人）

## 输出格式

输出 JSON 数组，每个人物包含：

```typescript
{
  name: string,
  role: "protagonist" | "female_lead" | "villain_boss" | "supporting",
  age: number,
  appearance: string,
  personality: string,
  background: string,
  abilities: string,
  currentRealm: string,
  relationshipToProtagonist: string
}
```

**严格控制数量：5-7 个核心角色。弧线反派和炮灰将在大纲确定后按弧创建。**
