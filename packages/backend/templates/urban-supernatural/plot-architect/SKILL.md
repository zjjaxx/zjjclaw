# 都市异能故事大纲设计

## 设计理念

**拒绝流水线式的"打怪升级"，追求让人放不下手机的故事。**

好故事的爽感不来自于"主角碾压一切"，而来自于**困局中的绝妙翻盘、选择时的两难煎熬、真相揭开时的恍然大悟**。读者记住的从来不是第几章打了谁的脸，而是那些让他们"卧槽"出声的瞬间。

核心原则：
- **信息差驱动**：最好的剧情张力来自角色之间的信息不对称——读者知道的比角色多（悬念感），或比角色少（惊奇感）
- **选择即代价**：每个重要决策都有代价，主角不能既要又要。获得力量要付出什么？保护一个人意味着放弃什么？
- **伏笔回收**：前期看似随意的细节，后期要变成关键转折点。让读者回头重读时有"原来如此"的快感
- **节奏呼吸**：不是一路高潮到底，而是张弛有度。高燃戏之后需要安静的角色互动来沉淀情感

## 节奏设计（替代机械化的"爽点密度"）

### 叙事节奏三层结构

```
微观节奏（每 3-5 章一个循环）：
  铺垫/日常 → 矛盾触发 → 紧张对峙 → 突破/反转 → 短暂喘息

中观节奏（每 15-25 章一个完整事件弧）：
  新谜题/新危机出现 → 调查/准备/选择 → 高潮对决 → 格局变化 + 新伏笔埋入

宏观节奏（故事弧线）：
  情绪曲线整体上扬，但每次上扬前要先下压（越挫越燃）
```

### 爽点不是"打脸"，而是"意料之外的精彩"

**真正的爽点类型**（替代无脑打脸）：

| 类型 | 说明 | 为什么爽 |
|------|------|----------|
| 布局揭晓 | 读者以为主角在吃亏，其实一切在掌握中 | 智商碾压比力量碾压更高级 |
| 绝境翻盘 | 不是靠外挂，而是靠之前埋下的伏笔/关系/牺牲 | 因为代价真实，所以胜利有重量 |
| 身份揭露 | 低调隐藏的实力/身份在关键时刻自然暴露 | 反差越大越爽，但要有合理理由 |
| 理念碰撞 | 两个都有道理的人正面交锋，用行动而非嘴炮分出高下 | 燃，因为双方都值得尊重 |
| 牺牲与回报 | 之前某个角色默默付出的代价，在此刻得到了回报 | 情感共鸣，催泪又热血 |
| 真相反转 | 以为理解的事情其实另有隐情，整个叙事角度翻转 | "卧槽"感，让人忍不住往回翻 |

**禁止的套路**：
- ❌ 对方先嚣张 → 主角沉默 → 一拳打飞 → 围观者震惊——这是2015年的写法
- ❌ 每个反派都是嚣张跋扈的脸谱，倒下后再也不出场
- ❌ 主角永远料事如神，每次都走最优解
- ❌ 用"实力碾压"解决一切问题

## 主角成长路径（替代线性升级）

### 三维成长模型

1. **能力成长**：境界突破不是打坐升级，而是"经历特定事件后的领悟"。每次突破都绑定一个关键剧情节点——可以是危机中的爆发，也可以是平静中的顿悟。突破应该有代价或副作用
2. **认知成长**：主角对世界的理解要不断被打碎重建。一弧以为的"真相"到二弧发现只是冰山一角。每次认知升级都伴随价值观的震荡
3. **关系成长**：从独来独往到信任他人、从被保护到保护他人、从个人恩怨到关乎更多人的命运。但成长不是线性的，会有"退步"——在受到背叛后重新封闭自己，然后再被拉回来

### 主角不是永远正确的

- 至少安排 2-3 次主角的**重大判断失误**，产生真实后果（不能靠运气化解）
- 主角要有"认知盲区"——由性格缺陷导致的思维死角，读者能看到但主角看不到
- 允许主角在某些事上被配角"教育"——不是降智，而是每个人都有自己的局限

## 动态弧线划分

根据 `targetChapters` 按比例计算各弧章节范围，严格按照以下比例切割：

| 总章节数 | 弧线数 | 各弧占比（依次） |
|----------|--------|-----------------|
| ≤ 100    | 2弧    | 40% / 60% |
| 101–200  | 3弧    | 20% / 35% / 45% |
| 201–350  | 4弧    | 15% / 25% / 35% / 25% |
| 351+     | 5弧    | 10% / 20% / 30% / 25% / 15% |

**计算规则**：章节范围 = 上一弧结束章 + 1 到 round(targetChapters × 累计占比)

用户 prompt 中会提供已计算好的弧线章节范围建议，直接采用即可。

## 各弧叙事任务

### 第一弧：暗流涌动

**核心目标**：建立世界观基底、让读者对主角产生共情、埋下贯穿全书的核心谜题。

**必须完成**：
- **悬念钩子**（前 3 章内）：不是"被欺负触发金手指"，而是一个反常事件打破主角的日常——可以是他发现自己一直信任的人在说谎，可以是某种异常现象只有他能感知，可以是他刻意隐藏的秘密即将暴露。要有**主动性**，主角不是被动挨打的受害者
- **金手指的代价**：异能/系统/传承的获得必须伴随某种限制或代价，而不是无条件的外挂。这个代价要成为贯穿全书的隐患
- **第一个两难选择**：逼主角在两个都不完美的选项中抉择，展现性格底色
- **核心谜题植入**：在日常中自然地埋入一个"不对劲"的细节——可以是主角身世、世界规则的漏洞、某个权威人物的矛盾行为。读者可能不会注意，但后期回看会恍然大悟
- **第一次能力展现**：不是碾压式的装逼，而是在困境中勉强获胜或用智取胜，让读者看到潜力而非实力

**本弧情感基调**：好奇、紧张、小有燃点，但更多是"水面下有东西"的不安感

### 第二弧：迷雾深处

**核心目标**：扩大格局、加深角色关系、让主角的认知被第一次打碎。

**必须完成**：
- **格局升级**：主角接触到更大的世界/组织/规则，发现一弧的事件只是冰山一角
- **关系深化**：角色之间不只是"队友"，要有独立于主角的互动——配角之间的矛盾、秘密、互助。至少一段让读者意外的关系变化
- **认知颠覆**：主角一弧中建立的某个认知/信任被打破（"我以为的好人其实......""我以为的规则其实有例外"）
- **主角失败**：安排一次不是靠运气能挽回的失败——失去某样东西（信任/关系/资源/时间），这个失去的代价要推动后续剧情
- **对手阵营的"合理性"展示**：通过某个事件让读者理解对手为什么这样做——不是洗白，而是展现另一个视角

**本弧情感基调**：复杂、纠结、有燃有虐，世界不再是黑白分明的

### 第三弧：破局之战

**核心目标**：矛盾总爆发、阵营重组、最大的反转出现。

**必须完成**：
- **多方博弈**：不是简单的"主角 vs 反派"，而是三方甚至多方势力角逐，每一方都有自己的算盘
- **内部裂痕**：主角团队或对手阵营出现严重分歧——不是背叛（太老套），而是"我们都想做对的事，但对的定义不同"
- **核心反转**：贯穿全书的某个"真相"被揭示，重新定义之前发生的事件。这个反转必须有充分的前期铺垫——回头能找到至少 3 个伏笔
- **高潮对决**：不只是战力比拼，而是理念/意志/牺牲的交锋。最好的高潮是"两个都对的人必须分出胜负"
- **代价真实**：胜利不是全员无伤，要有真实的失去——可以是关系的永久改变、某个角色的退场、主角自身的某种不可逆变化

**本弧情感基调**：燃到极致但底色苍凉，胜利中带着苦涩

### 后续弧线设计原则

如果有四弧或五弧：
- **不要简单地扩大地图**（"城市→省级→全国"这种游戏关卡式设计太机械）
- 而是**深化核心矛盾**——格局可以变大，但故事的核心始终是那几个人之间的选择和命运
- 后续弧的冲突要从"外部敌人"转向"内在矛盾"——当外部威胁解除后，团队内部的分歧才是最难处理的
- 每个新弧必须回答一个比上一弧更深的问题——从"谁是敌人"到"什么是正义"到"我到底想要什么"

## 感情线设计（替代机械化的百分比推进）

**感情不是进度条，而是化学反应。**

设计原则：
- **碰撞式推进**：两人之间的感情不是"好感度+1"式的线性增长，而是在冲突、合作、误解、坦诚中波浪式前进
- **独立叙事线**：感情线要和主线剧情有机交织——最好的感情戏发生在生死抉择中、理念冲突中、共同面对危机中，而不是独立的"约会章节"
- **双向变化**：不只是女主被主角吸引，主角也要因为女主而改变——改变某个偏见、正视某个弱点、做出某个不同的选择
- **张力保持**：即使关系确立后，仍然有张力来源——不是"情敌"这种外部威胁，而是两人价值观差异、成长方向不同带来的内在摩擦
- **拒绝的模板**：一见钟情、英雄救美、霸总式占有欲、"不许你受伤"式的爹味台词

## 结局设计

**好的结局不是"从此过上了幸福生活"，而是让人合上书后还在想。**

结局要求：
- **意料之外，情理之中**：最终的解决方式不能是读者一开始就能猜到的，但揭示后回顾全文要觉得"一切都指向了这里"
- **回应核心主题**：故事开始时提出的核心问题（关于力量/信任/成长/人性），要在结局给出一个不说教但有力量的回答
- **角色弧光完成**：主角的性格缺陷在结局中得到直面——不一定是克服，可以是接受、可以是与之共处
- **留白与余韵**：不要把所有线头都系得死死的，留一到两个开放性的暗示——让读者自己去想象后续。最好的结局是"故事结束了，但这些人的生活还在继续"
- **呼应开头**：结尾的某个场景/台词/意象要和开头形成对照——相似的场景，不同的主角，让人感受到变化的重量

**禁止的结局**：
- ❌ 主角成为最强者，所有敌人臣服
- ❌ 后宫收编，多条感情线并行开花
- ❌ Deus ex machina（靠从未提及的力量/角色解决最终问题）
- ❌ 纯粹的大团圆——胜利应该伴随某种永久的改变或失去

## 伏笔系统设计

每个弧必须包含**伏笔规划**：

- **当弧伏笔**：本弧埋入、本弧回收的小伏笔（1-2 个），用于单弧内的小反转
- **跨弧伏笔**：本弧埋入、后续弧回收的伏笔（2-3 个），用于长线叙事
- **全局伏笔**：从第一弧开始埋、最终弧回收的核心伏笔（1-2 个），是故事的灵魂

伏笔设计要求：
- 伏笔在埋入时必须是"自然的"——不能刻意突兀到让读者觉得"这里肯定有伏笔"
- 回收时要有"原来如此"的满足感——重新理解之前的事件/对话/行为
- 至少有一个伏笔是关于"某个角色的真实身份/动机"的

### newCharacterHints 设计要求

每弧的 `newCharacterHints` 应包含 3-6 条角色提示：
- **拒绝脸谱化**：不要"嚣张的富二代负责被打脸"这种。每个新角色都要有至少一个让人意外的层面
- **叙事功能明确**：每个角色存在的理由——推动主角成长/制造两难选择/揭示世界真相/搅动势力平衡
- **关系网络**：新角色和已有角色之间的关联要复杂化——不能只和主角有关系，要与其他核心角色也有交集
- 用已有核心角色的名字引用他们，新角色只给类型描述
- 每条提示写明：角色类型 + 叙事功能 + 与核心角色的关系
