# 玄幻修真弧线角色创建

## 核心原则

你正在为小说的**某一条故事弧**创建新角色。这些角色在本弧首次登场，主要在本弧发挥作用，部分可能延续到后续弧线。

**不要重复创建已有角色。** 你会收到已有角色列表，只需补充本弧缺少的角色。

## 本弧需要什么角色？

根据大纲提供的 `newCharacterHints`（角色提示），创建以下类型：

### 同辈对手/炮灰（cannon_fodder）：1-3个
本弧需要被打脸的角色：
- 宗门内的嚣张同辈/看不起主角的天才
- 秘境中争夺资源的对手
- 说话刻薄，喜欢当众羞辱，有靠山
- 具体的罪行（抢主角机缘/欺负主角朋友/公开羞辱）

### 弧线反派（antagonist）：1-2个
本弧的主要对手：
- 宗门长老中的阴谋者/敌对势力的高手
- 有自己的修炼目标和行事逻辑
- 修为在主角之上，需要主角突破才能击败
- 与大Boss可能有隐秘联系

### 弧线配角（supporting）：1-3个
本弧新出现的重要角色：
- 新地图的宗门掌门/城主/拍卖行主事
- 提供机缘/传功的神秘存在
- 新的友方角色（可能成为后续盟友）
- 本弧特定场景的关键角色

### 弧线女性角色（female_lead）：0-1个
如果大纲提示本弧有新的感情线：
- 有独立的修炼志向
- 与主角的关系要有独特性
- 不能和已有女主的类型完全重叠

## 与已有角色的关系网

创建新角色时，必须明确：
- 与主角的初始关系（敌对/中立/好感）
- 与已有核心角色的势力关联
- 在修炼体系中的定位（境界/门派/势力）

## 输出格式

```json
[
  {
    "name": "角色名",
    "role": "antagonist|cannon_fodder|supporting|female_lead",
    "age": 20,
    "appearance": "外貌描写（50字内）",
    "personality": "性格特征（100字内）",
    "background": "身世/来历（100字内）",
    "abilities": "修炼特长/法宝/特殊技能",
    "currentRealm": "当前境界",
    "relationshipToProtagonist": "与主角的关系",
    "firstAppearance": 1
  }
]
```

**每弧创建 3-6 个新角色。** 不要重复已有角色。
