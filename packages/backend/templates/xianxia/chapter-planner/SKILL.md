# 玄幻修真章节规划

## 章节节拍的基本结构

```
开头（200字内）→ 承接上章悬念
中段（1800-2000字）→ 3-5个清晰节拍
结尾（200-300字）→ 新悬念或爽点
```

---

## 节拍类型库（每章从中选3-5个）

| 节拍类型 | 触发条件 | 核心效果 |
|---------|---------|---------|
| **修炼推进** | 主角闭关/丹药突破/感悟天地 | 展示成长，铺垫实力 |
| **线索揭露** | 古籍/长辈暗示/遗迹发现 | 推进身世/金手指谜题 |
| **同辈摩擦** | 宗门日常/坊市偶遇/弟子聚集 | 累积打脸前置条件 |
| **打脸执行** | 比武/试炼/公开挑战 | 节奏高潮，情绪释放 |
| **秘境探索** | 进入禁地/争夺机缘 | 紧张感，资源获取 |
| **法宝/功法获取** | 遗迹/拍卖/传承 | 实力提升的物质基础 |
| **情感推进** | 偶遇女主/共同历险/误会 | 情感线推进 |
| **势力动态** | 宗门决议/大陆事件/威胁临近 | 宏观格局感 |
| **境界突破** | 积累到位，关键时刻突破 | 爽点高潮 |
| **危机降临** | 强敌追杀/天劫/阴谋揭露 | 悬念钩子 |

---

## 节奏控制原则

- **连续打脸上限**：最多连续3章处于打脸场景，之后必须插入修炼/情感/伏笔章
- **连续修炼上限**：最多连续2章写修炼场景，否则节奏迟缓
- **悬念类型轮换**：同一悬念类型（如"境界突破悬念"）不连续使用超过2次

### 高密度打脸章节结构
```
第N章：反派当众羞辱（刺激事件）
第N+1章：主角隐忍准备（积蓄情绪）
第N+2章：打脸触发，强势反转（释放）
第N+3章：余波处理，新敌人出现（引入下一段矛盾）
```

---

## 结尾钩子类型（必须具体，不能是概念）

| 类型 | 示例（具体场景/对话，≤50字）|
|------|--------------------------|
| 悬念钩子 | 他推开石室大门，地上的血迹还是新的。 |
| 反转钩子 | "你以为我的金丹只有一枚？" |
| 危机钩子 | 天际突然乌云翻滚——是天劫！ |
| 爽点钩子 | 在场所有人的目光同时凝固在那枚品阶标注上：**天级**。 |
| 情感钩子 | 她没有回头，但停顿了片刻，声音轻得像自言自语："傻子。" |

**endingHook 禁止写法：**
- ❌ "主角感到前途充满希望"（主题陈述，不是场景）
- ❌ "下一步主角要去秘境历练"（计划陈述，不是画面）

**beats 与 endingHook：** 最后一个 beat 必须与 `endingHook` 指向**同一终场画面/同一句台词**；不得在最后一 beat 里另写一套收束语再在 `endingHook` 里再写一套近义收束（会导致双结尾）。
