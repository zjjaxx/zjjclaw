# 玄幻修真力量体系设计

## 体系设计原则

好的修炼体系满足三个条件：
1. **可量化**：读者能感受到每个境界的战力差距
2. **有代价**：突破需要资源/机缘/感悟，不能靠努力无限叠加
3. **有上限焦虑**：主角当前境界在世界大局中处于什么位置

---

## 标准大境界划分（可根据世界观调整名称）

| 大境界 | 参考称谓 | 对应战力 | 突破核心条件 |
|--------|----------|----------|-------------|
| 一阶 | 练气期（1-9层）| 凡人巅峰→初入修炼 | 感知灵气，修炼功法 |
| 二阶 | 筑基期（初/中/后期）| 能使用法器，飞行 | 凝聚灵台，筑基丹辅助 |
| 三阶 | 金丹期（初/中/后期）| 宗门中坚力量 | 凝金丹，需机缘+量变 |
| 四阶 | 元婴期（初/中/后期）| 一方霸主，可统治一城 | 元婴出窍，小天劫 |
| 五阶 | 化神期 | 统御一国 | 大天劫，生死一线 |
| 六阶 | 合体期 | 大陆顶尖 | 感悟天地法则 |
| 七阶 | 大乘期 | 准飞升者 | 大道领悟，法则融合 |
| 八阶 | 渡劫期 | 飞升前夕 | 九九天劫 |
| 飞升后 | 真仙/上界 | 超脱凡界 | 飞升天劫 |

---

## 子境界细分

每个大境界分三小段（小三段制）：
- **前期**：刚突破，灵力不稳
- **中期**：境界稳固，战力提升
- **后期/圆满**：逼近下一境界，天地感应

战力公式参考：
- 同境界前期 vs 后期：差距约 1.5-2 倍
- 跨境界（低打高）：须有特殊法宝/功法/天赋加持
- 一般情况：小境界之差可抗衡，大境界（超过1阶）之差无解

## 主角金手指与体系结合

金手指必须与修炼体系有机结合，而非外挂：

| 金手指类型 | 与体系结合方式 |
|-----------|--------------|
| 上古神体/特殊灵根 | 修炼速度极快，可吸收同境界无法吸收的天材地宝 |
| 前世记忆/重生 | 知道秘境位置、功法等级、势力走向 |
| 系统（修真版）| 任务奖励→ 功法碎片、丹药、感悟加速 |
| 神秘传承 | 上古功法跳跃境界限制，或混合修炼路线 |
| 神魂强大 | 一魂化三宗，分身修炼 |

## 功法等级体系

- **凡级**：民间流通，上限筑基
- **玄级**：宗门传功，上限金丹
- **地级**：珍稀，上限元婴
- **天级**：千金难买，上限化神以上
- **仙级**：上界残页，威力无上
