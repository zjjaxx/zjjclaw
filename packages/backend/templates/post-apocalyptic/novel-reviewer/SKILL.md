# 末世重生章节审校

## 审校核心维度

### 1. 生存逻辑一致性
- 物资消耗是否合理（食物/子弹/药品有没有消失/凭空出现）
- 伤情是否被追踪（上章受伤，这章是否有处理）
- 空间储物清单是否前后一致

### 2. 重生记忆使用合理性
- 预知使用是否符合当前阶段的准确度（早期高准确，后期有偏差）
- 主角是否适当隐瞒记忆来源（未解释的先知行为越积累越好）
- 偏差事件是否引发了主角的内心变化

### 3. 末世氛围
- 是否保持了生存压力感（不能多章都是"很安全"）
- 物资/地点描写是否具体（不能漂浮）
- 人性灰度是否体现（全善/全恶都失真）

### 4. 节奏与爽感
- 重生者双轨叙事（外在行为+内心预知）是否到位
- 打脸布局是否在推进（当前悬挂对象多久没有结算）
- 章节结尾危机/爽点是否具体

### 5. 人物声音差异化
- 主角话少、精准，是否保持
- 各队员说话方式是否有差异

---

## 输出格式

给出：
1. **整体评分**（满分10分）
2. **优点**（2-3条，具体到段落）
3. **问题清单**（每条含：问题描述 + 修改建议）
4. **关键修改建议**（最需要改的1条，含改写示例）
