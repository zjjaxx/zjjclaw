# 末世重生金手指设计

## 末世金手指的本质

末世重生金手指 = **信息优势（重生记忆）** + **物资优势（空间异能）**

最好的末世金手指不是让主角无敌，而是：
- 让主角比别人**早一步**（但不是永远正确）
- 让主角在"最需要的时候"有资源支撑
- 让金手指的**局限性**成为剧情驱动力

---

## 核心金手指一：重生记忆

### 初始状态
- 主角在末日爆发前3天觉醒重生记忆
- 记忆完整度约60-70%（不是完美记录，有模糊区域）
- 上世存活到第X天，之后的事情一无所知

### 使用规则
- 地点类信息：最精准（物资点、安全路线）
- 事件类信息：有时间误差（早/晚几天发生）
- 人物类信息：可能偏差（上世某人是朋友，这世可能是敌人）

### 记忆偏差机制（剧情驱动引擎）

| 阶段 | 偏差程度 | 剧情功能 |
|------|---------|---------|
| 第1-30章 | 几乎无偏差（90%准确）| 建立主角"神准"人设，积累信任/怀疑 |
| 第31-80章 | 小偏差出现（70%准确）| 制造悬念，主角开始不能完全依赖记忆 |
| 第81-150章 | 重大偏差（50%准确）| 某关键事件完全不同，主角必须重新布局 |
| 第151章+ | 基本失效（只有大事轮廓）| 主角靠自己，成长完成 |

### 记忆偏差的原因设计
- 主角这世的改变已经影响了历史（蝴蝶效应）
- 某些人因为主角的介入做出了不同选择
- 上世记忆本身不完整（极度恐惧时的记忆会有空白）

## 核心金手指二：空间异能

### 初始状态
- 末日前一天觉醒（比所有人早，完全靠重生记忆指引）
- 初始容量：约5立方米
- 初始功能：储物（取放物品，不受重量限制）

### 成长阶段

| 进化节点 | 容量 | 新增功能 | 触发条件 |
|---------|------|---------|---------|
| 初始 | 5立方米 | 储物 | 末日前觉醒 |
| 二阶突破 | 30立方米 | 储物+瞬间取放 | 二阶晶核吸收 |
| 三阶突破 | 100立方米 | +5米内位移 | 极限战斗中觉醒 |
| 四阶突破 | 500立方米 | +切割空间（武器化）| 生死危机中爆发 |
| 五阶突破 | 无限（理论）| +远程传送（短距）| 终局觉醒 |

### 使用限制与代价
- 空间使用消耗精神力，高频使用会导致眩晕/昏迷
- 空间内时间静止（物品不会腐烂），但取放需要意识集中
- 不能将活人放入空间（生命体放入会致死）——重要限制！
- 高阶功能（切割/传送）使用代价极大，不能连续使用

### 空间异能的多场景活用
- **战斗**：瞬间换武器、从空间取出各种物品当武器
- **侦查**：在空间里放入摄像头/传感器，形成情报网络
- **建设**：从空间取出预储的建材，快速改造据点
- **贸易**：主角是末世最大的"行走超市"，贸易影响力巨大

---

## 两个金手指的协同策略

| 情境 | 两金手指协同 |
|------|------------|
| 提前抢占物资 | 重生记忆→知道物资点；空间→一次清空 |
| 避免危机 | 重生记忆→知道陷阱；空间→装备应对工具 |
| 人性考验 | 记忆告诉主角"这个人上世背叛了"；但空间物资帮主角维持团队不依赖那个人 |
| 记忆失效后 | 没有预知，但空间物资积累已经建立了最大的物质优势 |
