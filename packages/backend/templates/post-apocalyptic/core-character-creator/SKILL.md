# 末世重生核心人物创建

## 核心人物定义

核心人物 = 从末日初期到终局决战都有戏份的角色。

**只创建以下角色：**

| 类型 | 数量 | 说明 |
|------|------|------|
| protagonist | 1 | 重生主角 |
| female_lead | 1-2 | 核心女主（全书羁绊，非某一弧过客） |
| villain_boss | 1 | 终极仇敌（上世害死主角的人/末世最大威胁） |
| supporting | 2-3 | 核心队友（全书并肩作战） |

**总计 5-7 个角色**。各据点NPC、弧线反派、临时盟友将在大纲确定后按弧创建。

## 主角设定

**表面设定：**
- 末日前：普通人（大学生/白领/小生意人）
- 末日后：异能觉醒早，行动处处超前
- 外在特征：沉稳、提前准备、不显露能力来源

**内在设定（重生者的内心世界）：**
- 上一世的遗憾（死于信任了错误的人/没保护好某人）
- 这一世的决心（弥补、复仇、或单纯活下去）
- 重生者的孤独感（知道未来却无法全说出口）

**行为特征（打脸伏笔）：**
- 总在"正确时机"出现在正确地点
- 能预判危险但给的理由模糊
- 对某些人第一眼就远离/接近，后来证明判断正确

## 女主设定

核心女主要在末世中有独立生存能力和目标：
- 与主角的羁绊来自生死共患难
- 对主角的"先知行为"既信任又好奇
- 有自己的异能成长线

## 终极仇敌设定

- 上一世害死主角/队友的人，或末世秩序的最大破坏者
- 前期可能伪装为盟友/中立方
- 有令人信服的生存逻辑（不是纯粹的恶）
- 实力/势力远超主角当前阶段

## 核心队友设定

- **忠实伙伴**：力量/速度系，直爽讲义气，第一个信任主角的人
- **治愈者/后勤**：团队不可或缺的支撑角色
- **智谋型**：多疑聪明，对主角起疑但最终成为最可靠盟友

## 输出格式

```json
[
  {
    "name": "角色名",
    "role": "protagonist|female_lead|villain_boss|supporting",
    "age": 24,
    "appearance": "外貌描写（50字内）",
    "personality": "性格特征（100字内）",
    "background": "身世/来历（100字内）",
    "abilities": "异能/技能描述",
    "currentRealm": "当前异能等级",
    "relationshipToProtagonist": "与主角的关系"
  }
]
```

**严格控制 5-7 个核心角色。各弧的据点人物、敌对势力、临时盟友将在大纲确定后按弧创建。**
