# 末世重生世界观构建

## 核心设计原则

末世世界必须同时提供：
1. **生存张力**（食物/安全/信任的持续匮乏）
2. **力量成长空间**（异能觉醒+进化，让战斗有意义）
3. **人性灰度**（最大的威胁往往不是丧尸，而是人）

---

## 必须构建的世界要素

### 1. 末日时间线

设计从末日第一天到主角故事开始的关键节点：
```
Day 0：病毒/事件触发，第一批感染者出现
Day 1-7：城市失控，政府失能，人口大规模流失
Day 8-30：幸存者自发聚集，初级据点形成
Day 31-180：势力分化，军队/政府残余/民间自发组织各霸一方
Day 181+：稳定的末世秩序初步形成（故事通常从这里开始）
```

### 2. 丧尸进化体系

丧尸不能一成不变，须有进化威胁：

| 丧尸阶段 | 特征 | 对人类的威胁 |
|---------|------|------------|
| 普通丧尸（0阶）| 缓慢，只会咬人 | 群体危险，单体可处理 |
| 进化体（1-3阶）| 特殊能力（装甲/毒素/速度/力量） | 需要异能者才能安全处理 |
| 王者丧尸（4-5阶）| 有初步智能，可指挥普通丧尸 | 区域级威胁 |
| 变异领主（6+阶）| 高智能，巢穴建立，资源据点 | 大规模军事行动才能清除 |

丧尸晶核（水晶）是进化的核心资源：
- 普通丧尸→低阶晶核，可辅助人类异能觉醒/进化
- 高阶丧尸→稀缺资源，各方势力争夺目标

### 3. 异能觉醒机制

- **觉醒比例**：约1%的人类能觉醒异能（稀缺性制造价值）
- **觉醒触发**：极度恐惧/绝境/进食晶核
- **异能类型**：火/冰/雷/空间/植物/心灵/治愈/增幅力/速度等
- **等级划分**：一阶觉醒→九阶超凡（每3阶一个大台阶）
- **进化方式**：吸收同系晶核/极限战斗/功法引导

### 4. 幸存者势力格局

三类主要势力：
- **军事基地**：强组织性，资源集中，专制倾向
- **民间自治据点**：凝聚力靠情感和规则，异能者话语权大
- **散修/小团伙**：机动灵活，高风险高回报

势力间的核心矛盾：
- 物资/粮食争夺
- 异能者归属（是共有资源还是私人力量）
- 领土扩张与防守

### 5. 资源体系

| 资源类别 | 稀缺程度 | 获取方式 |
|---------|---------|---------|
| 食物/净水 | 极稀缺 | 物资库/超市/农业种植（后期）|
| 丧尸晶核 | 稀缺 | 击杀丧尸 |
| 药品/医疗 | 极稀缺 | 医院/药店/黑市 |
| 武器/弹药 | 稀缺 | 军事仓库/黑市交易 |
| 主角空间物资 | 充裕（金手指）| 末日前囤积/秘密获取 |
