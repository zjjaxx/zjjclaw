# 末世重生弧线角色创建

## 核心原则

你正在为小说的**某一条故事弧**创建新角色。这些角色在本弧首次登场，部分可能延续到后续弧线。

**不要重复创建已有角色。** 你会收到已有角色列表，只需补充本弧缺少的角色。

## 本弧需要什么角色？

根据大纲提供的 `newCharacterHints`（角色提示），创建以下类型：

### 弧线反派（antagonist）：1-2个
本弧的主要对手（末世反派不是纯粹的恶人，而是"选择了不同生存方式的人"）：
- 专制军头：用强权保护弱者（自认为）
- 伪善领袖：表面给安全感，实则剥削
- 高阶异能者：有能力便有特权
- 利益集团头目：控制资源 = 控制人

### 炮灰/被打脸者（cannon_fodder）：1-2个
本弧需要被打脸的角色：
- 质疑主角决策的人（事后证明主角对）
- 仗着异能欺负普通人的人
- 背叛团队信任的人

### 弧线配角（supporting）：1-3个
本弧新出现的重要角色：
- 新据点/势力的领导者
- 掌握关键资源/情报的人
- 新队员候选人
- 关键的幸存者NPC

### 弧线感情角色（female_lead）：0-1个
如果大纲提示本弧有新的感情发展：
- 末世中独立生存的女性
- 与主角在共患难中建立信任
- 有自己的异能成长线

## 末世角色设计要点

- 每个角色都要有明确的**生存逻辑**（为什么还活着？靠什么？）
- 异能等级要合理（与当前时间线匹配）
- 角色间的**信任/猜忌**关系要清晰
- 反派的动机要可理解（在末世里任何选择都可能有合理性）

## 输出格式

```json
[
  {
    "name": "角色名",
    "role": "antagonist|cannon_fodder|supporting|female_lead",
    "age": 28,
    "appearance": "外貌描写（50字内）",
    "personality": "性格特征（100字内）",
    "background": "身世/来历（100字内）",
    "abilities": "异能/技能描述",
    "currentRealm": "当前异能等级",
    "relationshipToProtagonist": "与主角的关系",
    "firstAppearance": 1
  }
]
```

**每弧创建 3-6 个新角色。** 不要重复已有角色。
