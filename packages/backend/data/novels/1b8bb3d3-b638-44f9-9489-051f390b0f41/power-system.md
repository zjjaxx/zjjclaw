# 《上海城隍》神道力量/修炼体系设计

## 一、体系核心：“神箓”驱动，权柄为基

本世界力量体系不称“修炼”，而尊“**持箓**”或“**奉道**”。力量根源并非单纯吸纳灵气，而是以**天道认可的神箓为核心**，调动、整合、敕令三种力量：
1.  **香火愿力**：来自信徒供奉、百姓感念、一方水土的认可。最普遍但驳杂，需神箓提纯。
2.  **功德之力**：源于维护阴阳秩序、庇护生灵、消弭灾祸所得的天道反馈。纯净稳固，是构建神职法力的基石。
3.  **天地灵气**：末法时代已极为稀薄，但某些风水宝地、古迹节点仍有残存。可辅助淬炼神躯、施展法术。

**主角金手指特殊性**：主角秦川被**天道直接授箓**，获得“上海都城隍”神箓，这意味着：
*   **正统性最高**：对一切依托上海地脉、民俗信仰存在的野神、地祇、妖鬼具备先天位格压制。
*   **成长捷径**：无需像散修或家族传承者那样苦苦摸索、积累香火、祈求“上天感应”，他的神箓本身就是“许可证”和“加速器”。
*   **权柄清晰**：神箓自带基础权柄信息（虽需逐步解锁），指明了力量发展的正统方向。
*   **微型庙宇空间**：这是独属于“授箓正神”的福利，是神域的雏形，可隔绝窥探、加速炼化香火功德、储存神力、乃至后来培育属神/阴兵。

*每阶细分“初入、稳固、圆满”三个小层次。*

## 三、各阶战力详述与晋升方式

### **第一阶段：凡 → 神（一至三阶）**
*   **战力表现**：
    *   **一阶**：力量集中于庙宇。能让人走进城隍庙感到心安神宁，能让普通鬼魂无法靠近。现实战斗力不强，但“位格”高，对非正统鬼怪有震慑。
    *   **二阶**：可短途“神游”，以灵体状态巡查附近街区。能施展“破邪金光”（微弱）、“安魂咒”。武力可击溃水泥墙，速度超常人。
    *   **三阶**：可短时间（半小时）凝出神力化身在外行走，调用地气形成简单结界（如“画地为牢”）。能初步使用城隍印（虚影）砸鬼，威力如小轿车撞击。
*   **晋升条件**：
    *   **积累**：主要依赖**香火愿力**（来自解决小规模灵异事件、庇护乡里）和**微末功德**。
    *   **关键**：对神职权柄的**理解与践行**。越是履行城隍职责（如调解邻里阴气纠纷、送滞留亡魂入阴），天道反馈（功德）越多，晋升越快。
    *   **主角优势**：微型庙宇空间可高效提纯、储存香火，加速这一积累过程。神箓自带指引，避免走弯路。

### **第二阶段：神 → 地祇（四至六阶）**
*   **战力表现**：
    *   **四阶（神庭境）**：质变点。可在辖区内（如老城厢）构建“临时神域”，领域内感知大幅增强，可小范围改变环境（如让迷雾弥漫、让灯火通明）。可召唤历史中的“上海县阴兵”虚影助阵。
    *   **五阶（显圣境）**：神威实质化，可形成威压让低阶修行者心神震颤。能施展“神罚·雷霆”（小范围）、“律令·禁锢”。可册封1-3名游巡阴兵（需有合适阴魂）。
    *   **六阶（法相境）**：终极地祇。法相显化，高达数丈，万民可见（但普通人会自我合理化为“海市蜃楼”或“光学现象”）。可发动“城隍审判”，直接剥夺辖区内邪神、大妖的部分力量根源。能短暂沟通上海的地脉龙气（若有设定）。
*   **晋升条件**：
    *   **海量资源**：需要磅礴的香火（至少一个稳定街区或特定群体的持续信仰）和重大功德（解决影响成千上万人的灵异灾难、维护重大历史遗迹灵脉）。
    *   **核心领悟**：领悟“神与城的关系”、“秩序与庇护的真意”。需要举行或参与**大型仪式**（如修复城隍庙古阵、主持清明或中元节大型祭奠）。
    *   **关键道具**：可能需要寻回“上海都城隍”的**旧印信**、**法袍**等遗物，或获得关键历史人物的认可（英灵）。
    *   **主角优势**：天道正统，在获取“城市认可”方面有加成。庙宇空间可成长为“训练营”和“后勤基地”，储存的香火神力是关键储备。

### **第三阶段：地祇 → 天神（七阶及以上）**
*   **此为传说境界**，是故事长期目标。晋升需**与上海共命运**，解决城市级乃至国家级的灵异危机（如抵御上古魔头复苏、修补黄浦江下的“海眼”裂隙），获得亿万民众的无意识感念（城市集体潜意识认可），并最终将自身神道法则与城市发展脉络融合。

## 四、资源体系

1.  **香火愿力**：最基础资源。来源：
    *   **主动庇护**：解决辖区灵异事件，获得当事人及其亲友感念。
    *   **庙宇供奉**：城隍庙内信徒上香、许愿、还原产生的愿力（需提纯）。
    *   **城市认同**：因维护城市安宁、文化传承，获得部分市民的集体潜意识认可（量少但质纯）。
2.  **功德之力**：高级资源。来源：
    *   **维护阴阳秩序**：引渡亡魂、镇压厉鬼、调解人鬼纠纷。
    *   **庇护生灵**：在灾难中保护凡人。
    *   **修复灵脉节点**：修复如“龙柱”、“古刹阵眼”等风水/封印节点。
3.  **天地灵气/天材地宝**：
    *   **灵气节点**：静安寺、玉佛禅寺、豫园等古建筑群；外滩万国建筑群的风水交汇点；苏州河某些故道。
    *   **天材地宝**：百年槐木心（通阴）、古井寒玉（镇魂）、沾染历史人物气息的古物（承载特殊愿力）、异兽精血（罕见）等。
    *   **法脉古籍**：秦家祖传的残卷、从其他修行者或“特民办”库房交换获得的功法、仪轨记载。
4.  **特殊资源**：
    *   **神职配件**：遗失的城隍印、令旗、法典等，可大幅增强权柄。
    *   **阴兵鬼差**：收复、培养或“召唤”的灵体下属，本身是战力也是消耗品（需香火维持）。
    *   **秘境/遗迹**：上海地下可能存在的古遗址、通往“水府”或“阴司临时办事处”的通道，内有险阻也有机缘。

## 五、特殊能力种类（世界填充）

本世界能力体系驳杂，但可归为四大类，主角的“神道系”位列顶端：

1.  **神道系（正统/核心）**：
    *   **代表**：主角（城隍）、其他地区正神（稀少）、有编制的阴神。
    *   **特点**：力量源于神箓与天道权柄，中正平和，克制邪祟。能力与职责绑定（如城隍的“监察”、“审判”、“庇护”）。
    *   **常见能力**：神威压制、敕令术法（言出法随弱化版）、构建神域、驱邪破煞、沟通阴阳。

2.  **民俗法脉系（民间主流）**：
    *   **代表**：秦家祖传风水术（已残）、走阴人、赊刀人、鲁班术传人、出马仙（上海较少）、“沪渎龙宫”水法。
    *   **特点**：依托地方民俗、行业信仰、家族血脉传承。威力不俗但往往有缺陷或代价（如五弊三缺、特定禁忌）。与神道系有合作也有竞争。
    *   **常见能力**：风水堪舆、符箓阵法、请神上身（请野神或祖先）、驭使精灵（水鬼、黄仙等）、诅咒祝由。

3.  **诡异/异变系（混乱/反派常客）**：
    *   **代表**：怨灵厉鬼、山精野怪、因执念或巧合获得力量的“怪诞”（如都市传说实体化）、修炼邪法的修行者。
    *   **特点**：力量来源负面（怨气、恐惧、血肉祭祀），成长迅速但易失控，被神道系和正统法脉克制。能力千奇百怪。
    *   **常见能力**：幻术恐惧、操控阴影、血肉寄生、领域污染、规则类诅咒（如必须在听到某种声音后回头）。

4.  **秘术/外来系（变量）**：
    *   **代表**：西方秘法学会（元素、召唤）、东南亚降头师、日本阴阳师世家、依托现代科技的“赛博术士”（极稀有）。
    *   **特点**：体系迥异，规则不同，常带来意外与冲突。是上海“海纳百川”暗面的体现。
    *   **常见能力**：元素操纵、契约召唤、式神驱使、数据灵魂投影（设想）、跨文化诅咒与祝福。

## 六、主角金手指具体加成规则

1.  **神箓（天道U盘）**：
    *   **效率倍增**：香火、功德炼化效率是同等境界民俗修行者的**5-10倍**。
    *   **权限解锁**：随着境界提升，自动解锁相应权柄的使用方法（如三阶解锁“监察”，四阶解锁“引渡”）。
    *   **位格压制**：对上海地域内所有非天道正封的灵体、妖鬼、野神，自带1-3个小境界的威压优势。

2.  **微型庙宇空间（成长型神域）**：
    *   **初始状态**：仅一间大殿大小，时间流速略快于外界（1.2:1），可储存物品与神力。
    *   **成长性**：随境界提升和功德注入而扩大。三阶可出现厢房（可让灵体暂居），四阶出现院落（可训练阴兵），六阶出现山水园林（模拟上海地貌）。
    *   **核心功能**：
        *   **修炼加速**：空间内炼化香火无损耗。
        *   **绝对安全区**：可肉身进入（需消耗神力），躲避绝大部分探测与追杀。
        *   **资源生产**：后期可引入灵泉、灵植，或凝聚“香火钱”、“功德米”等特殊资源。
        *   **神域投射**：高境界后，可将空间的部分规则（如禁法、重力）短暂投射到现实。

此体系将主角的成长与上海的城市变迁、民俗网络深度绑定，每一阶段的突破都伴随着对城市秘密的更深挖掘和职责的加重，完美支撑《上海城隍》的剧情推进与世界观展开。
