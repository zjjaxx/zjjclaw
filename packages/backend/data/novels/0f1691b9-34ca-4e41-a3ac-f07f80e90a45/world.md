# 《科学修仙传》世界观设定：云渺洲

## 大陆格局

**大陆名称**：云渺洲，状如浮空巨叶，东西横跨百万里，南北纵横七十万里。四周为无尽**星陨海**所环绕，海上常年弥漫能消融灵识的“噬灵迷雾”，唯有少数稳固的“天舟航线”连接着零星海外仙岛。

**灵气分布与核心矛盾**：
- 灵气并非均匀分布，而是呈“灵脉网络”形态。九条**主灵脉**如同大陆骨架，其交汇处便是顶级宗门与上古遗迹所在。
- 灵气本质为本世界未解之谜，传统认知为“天地造化之气”。**主角的科学视角将首次提出假说：灵气可能是一种具有特殊波粒二象性、能与生命场和精神力共振的高维能量粒子（暂称“灵子”）**。灵脉则是天然的“灵子”高浓度富集与定向流动通道。
- **凡人与修士**：壁垒森严又相互依存。修士超然物外，掌控资源与天道法则；凡人亿万，构成社会基底，为宗门提供弟子来源与基础物资。凡人王朝多为地级、县级宗门的附庸或代理。修士不在凡俗轻易显露神通，是为“天道禁约”，以免因果缠身，但高阶修士一念可决王朝更迭。

## 宗门势力分布

势力呈清晰的金字塔结构，遵循弱肉强食的修真法则。

1.  **天级势力（执棋者，2个）**：
    - **天衍剑宗**：位于**中域·万剑山脉**主灵脉核心。奉行“一剑破万法”，认为剑道是直达天道的最优解。对异端道统（尤其是试图解析天道根本者）抱有敌意。
    - **太上丹宫**：位于**东域·药王原**主灵脉核心。垄断高阶丹药，坚信“丹道通玄”，以丹药调和阴阳、逆天改命。对资源控制欲极强。

2.  **地级势力（角逐者，约十二家）**：
    - **青云门**（东域，依附太上丹宫）
    - **玄阴教**（北域荒原边缘，魔道，擅炼魂驭鬼）
    - **星罗阁**（西域，中立，擅长阵法与星象推演）
    - **百炼宗**（南域，中立，擅长炼器）等。
    - **势力动态**：地级势力在天级宗门的阴影下合作、竞争、倾轧，争夺灵脉支脉、秘境名额与天才弟子。

3.  **县级势力/散修（求生者）**：
    - 掌控一城一地的小宗门或修真家族，仰仗地级势力鼻息。
    - 散修联盟“四海堂”等组织，在夹缝中寻求资源，信息混杂。

4.  **禁忌势力（世界暗面）**：
    - **古修遗族“渊族”**：传说居住于地底灵脉深处或某些绝地，掌握着与现代修真体系迥异的古老力量，视当前修真文明为“歧路”。
    - **上古妖族“十万大山”**：占据南域无尽山脉，保有先天血脉神通与古老传承，对人类修士警惕且敌视，是其材料（妖丹、精血）的潜在来源，也是极度危险之地。

## 主要禁地与秘境

1.  **禁地**：
    - **坠星渊**（中域与西域交界）：上古天外陨星坠落之地，终年笼罩狂暴的磁元罡煞与紊乱灵气流，法宝失灵，灵识错乱。**疑似蕴藏不属于本世界的奇异金属与能量辐射（主角科学实验的潜在高风险材料区）**。
    - **无归海眼**（星陨海深处）：巨大的海洋灵气漩涡，疑似通往其他界域或上古战场碎片的空间裂缝，进入者从未归来。
    - **寂灭雷泽**（北域）：永恒雷暴区，盛产雷属性至宝，但天雷无常，乃天然渡劫场，亦为炼体绝地。

2.  **秘境**：
    - **五行小洞天**（每甲子（60年）开启）：最著名的公共秘境，内部分布金木水火土五种极端环境区域，出产相应属性的稀有灵材。进入需“洞天符”，为各大势力争夺焦点。
    - **天工秘境**（每三百年开启，下次开启在百年后）：传说为上古“天工族”遗留的试炼之地，内有大量残缺的古代傀儡、机关与奇特种类的材料。**此秘境是主角运用工程学知识，修复并反向研究古代灵能科技的最佳场所**。

3.  **上界通道**：
    - 飞升传说确存在，但已断绝近万年。记载中，需渡过渡劫期最后一重“飞升天劫”，接引仙光自九天垂下，方可打开通道。如今，飞升已成传说，最高战力止步于大乘期。**有隐世研究者猜测，可能是世界“灵子”循环系统出现了未知故障或封锁**。

## 天地法则与修炼资源

- **天劫**：金丹晋元婴、化神晋炼虚、大乘期飞升，共三次大天劫。形制主要为雷劫，但内含心魔拷问。**科学视角猜想**：天劫或是世界“灵子场”对个体过于强大的“灵子聚合体”（修士）产生的系统性排异反应与淬炼机制。
- **道韵与修炼路线**：主流为剑修（极致攻击）、丹修（资源转化）、器修（外物依仗）、阵修（借用天地）、法修（灵力多变）、体修（开发自身）。**主角的科学修真路**将被视为前所未有的“异数”，初期会被贬低为“奇技淫巧”或“匠气之道”。
- **资源体系**：
  - **基础**：下品→中品→上品→极品灵石（灵子凝结体）。
  - **材料**：灵草（需符合生长周期与生态）、矿物（蕴灵金属/玉石）、妖兽材料（特定部位）。
  - **法宝**：法器→灵器→法宝→灵宝→通天灵宝。
  - **核心逻辑**：灵脉即能源命脉。大宗门垄断高阶知识（功法、丹方、阵图）与核心材料产地。坊市流通中低层资源，顶级拍卖会出现稀有物。**主角的优势在于能通过成分分析、性质测试与合成实验，发现现有材料的非传统用途，甚至利用科学方法（如电解、高温高压合成、提纯）制备出“人工合成灵材”（失败率与风险并存）**。

## 主角起点所在地

**地点名称**：**青木城**，隶属东域“青云门”势力范围的边缘地带。

**详细描述**：
- **地理与环境**：坐落于一条小型“木属性”灵脉的末梢分支上，周边丘陵盛产低阶灵木与草药。灵气浓度稀薄，仅能支撑筑基期修士日常修炼，金丹修士在此会感觉“灵气饥渴”。
- **势力构成**：
    1.  **青木城赵家**（主导）：县级修真家族，家主为筑基后期修为，依附于青云门外门，负责为青云门定期收集特定灵草与初炼矿物。
    2.  **青木城李家**（竞争）：另一筑基家族，与赵家明争暗斗，主要经营低阶符箓生意。
    3.  **散修与小作坊**：城中存在少量炼气散修，以及为修士服务的“百工坊”（低劣法器修理、符纸制作、草药初加工）。
- **社会状态**：凡人占九成以上，知晓修士存在并敬畏，生活与此界凡人王朝无异。城内有一处小型“修士坊市”，每月初一、十五开放，交易低阶物品。
- **作为起点的意义**：
    1.  **信息与资源匮乏**：高阶知识被封锁，主角的“科学理论”在此地如同天书，缺乏验证和实践的高阶素材。他必须从最基础的本地材料（灵木、常见矿物、低阶妖兽材料）开始实验，失败是常态。
    2.  **相对安全与低调**：地处偏远，高手罕至，适合主角暗中进行初期、可能引发小型爆炸或奇怪现象的“不靠谱”实验。
    3.  **冲突与机遇的起点**：赵、李两家的争斗，坊市中的信息碎片，家族收藏的残缺古籍，以及城外山林中可能存在的未被充分认知的独特材料（例如，某种具有特殊晶体结构的矿石，或灵力传导率异常的植物），都将成为主角运用科学思维，撬动第一个支点的舞台。
    4.  **第一道障碍**：主角初期最大的困难并非敌人，而是 **“认知壁垒”** 和 **“材料壁垒”** 。他需要将物理、化学、生物知识，艰难地与本世界的“灵气参数”结合，并找到或制备出能实现其设想的“灵导材料”、“催化介质”或“能量核心”。每一次成功，都将是对云渺洲修真基础理论的微小却坚实的颠覆。
