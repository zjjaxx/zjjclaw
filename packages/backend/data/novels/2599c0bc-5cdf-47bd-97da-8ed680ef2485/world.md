# 《重生仙尊：回到修为尽废那一天》世界观设定

## 大陆格局

**天玄大陆**，广袤无垠，生灵亿兆，是为修真主舞台。大陆主体由**中土神州**、**东极云海**、**西漠荒土**、**南离万泽**与**北原冰域**五大疆域构成。大陆之外，**无尽海**环绕，中有星罗棋布的**海外仙岛**；北方冰原之外，更有空间紊乱、煞气冲天的**上古战场遗址**，乃古修与天魔决战之地，至今为绝域。

**灵气分布**极不均匀。大陆之下蕴藏九条巨型**祖脉**，其延伸出的灵脉网络决定了世界的繁华与荒芜。中土神州独占其四，为天下灵气核心，亦是顶级宗门必争之地。其余疆域灵脉逐级递减，至西漠、北原，灵气稀薄，资源匮乏，多为苦修者或流放者所居。

**凡人与修士**之间，存在一道无形而坚固的鸿沟。修士超然物外，掌控伟力，凡人王朝多为地级乃至玄黄级宗门的附庸，负责提供有灵根的弟子与基础资源。修士在凡人面前现身并非禁忌，但高阶修士通常隐于洞天福地，凡人终其一生难见真容。两者生活世界虽有交集，但地位天差地别。

## 宗门势力分布

天下势力，等级森严，呈金字塔结构：

*   **天级宗门（2个）**：立于大陆顶点，掌控祖脉源头，门内有渡劫期老祖坐镇，遥不可及。
    *   **青冥剑宗**：位于中土神州**天剑山脉**，以剑修闻名天下，主张“一剑破万法”，门风凌厉，护短。
    *   **玄天宗**：位于中土神州**问道峰**，道法传承最为古老正统，丹、阵、器、符四道底蕴深不可测，被誉为“万法之源”。
*   **地级宗门（10-20个）**：占据主要灵脉，拥有化神期以上强者，是一域霸主。彼此间合纵连横，竞争激烈。
    *   **代表**：**药王谷**（丹道）、**神兵阁**（炼器）、**万阵山**（阵法）、**驭灵宗**（御兽）等。
*   **玄黄级宗门/散修组织**：数量众多，占据中小型灵脉或依附于地级宗门。金丹期为中坚，元婴可称老祖。
*   **凡人王朝**：依附于附近宗门，皇室往往与修真家族有千丝万缕的联系。

**特殊势力：**
*   **魔道魁首——血煞宗**：盘踞于**西漠**与**南离**交界处的**血狱渊**，行事诡秘狠辣，与正道宗门长期敌对，但实力强横，无人敢轻言剿灭。
*   **中立圣地——瑶光仙宫**：位于**东极云海**深处的**浮空仙岛**，只收女弟子，超然于正魔之争，以情报和高端拍卖会闻名，底蕴神秘。
*   **禁忌势力一：古巫遗族**：隐居**南离万泽**最深处，传承上古巫术与图腾之力，视外界修士为敌，其领地乃生灵禁区。
*   **禁忌势力二：战魂宗**：活动于**北原**及**上古战场遗址**边缘，修炼诡异魂道，能驱策古战魂，探寻战场遗宝，亦正亦邪，令人忌惮。

## 主要禁地与秘境

*   **上古战场（绝地）**：大陆北方终极禁地。空间破碎，法则混乱，充斥着古战场遗留的杀意、煞气以及陨落大能的残缺执念。化神期以下入内十死无生，但据说埋藏着上古仙魔的传承与神兵。偶尔有空间裂隙将内部物品抛出，引发腥风血雨。
*   **万妖山脉（凶地）**：横贯**南离万泽**的巨型山脉，是妖兽、灵兽的乐园，深处有化形妖王统领，人类修士禁入核心区。外围是低阶弟子试炼、寻找灵材的主要场所。
*   **轮回谷（奇地）**：位于**中土神州**西部，每甲子（六十年）开启一次。谷内时间流速异常，遍布幻境与轮回法则碎片，是修士锤炼心境、感悟轮回之道的珍贵机会，但沉沦幻境者亦会神魂俱灭。
*   **瑶光秘境（福地）**：由**瑶光仙宫**控制的顶级秘境，每三十年开放一次，仅限百岁以下金丹修士进入。其中天材地宝无数，更传闻有直指大道的传承，是天下年轻天才角逐的最高舞台之一。
*   **飞升台（传说）**：位于大陆未知处，是连接上界的唯一通道。但自上古大战后受损，已有万年无人成功飞升。修复飞升台，打通仙路，是此界所有巅峰修士的终极梦想与执念。

## 天地法则与修炼资源

**修炼体系与天劫**：
*   主流境界：**炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫**。
*   天劫：自**元婴**破**化神**起，每突破一个大境界，皆需渡劫。化神为**三九雷劫**，炼虚为**六九雷劫**，合体为**九九雷劫**，大乘至渡劫更有恐怖的**心魔劫**与**混元劫**。渡劫期需历经九重天劫，方有望触摸飞升之门。

**修炼路线**：
*   **剑修**：攻伐第一，一剑破万法，但需极端专注，资源消耗亦巨。
*   **法修**：道法博大，手段繁多，依赖深厚灵力与道法领悟。
*   **丹修/器修/阵修/符修**：依托外物，前期战力较弱但地位崇高，资源获取能力强。
*   **体修**：锤炼肉身，近战无敌，修炼过程极端痛苦，需大量气血宝物。
*   **魂修/鬼修/巫修等**：偏门路线，威力奇诡但隐患多，常被视为邪道。

**资源体系**：
*   **货币/能量**：下品灵石 → 中品灵石 → 上品灵石 → 极品灵石（蕴含微量法则）。更高层次交易以物易物，或使用**灵晶**（灵石矿脉核心）。
*   **天材地宝**：凡品 → 灵材 → 宝药（对应金丹以下）→ 玄药/玄铁（元婴化神）→ 神药/神料（炼虚以上）→ 道源圣物（传说，蕴含完整法则碎片）。
*   **法宝**：法器 → 灵器 → 法宝 → 古宝/灵宝 → 通天灵宝 → 玄天之宝（蕴含一缕天道法则）。

## 主角起点所在地（详细描述）

**地点**：天玄大陆，**东极云海**边缘，**离国**境内，地级宗门**青岚宗**。

**青岚宗概况**：
*   **地位**：东极云海边缘区域三大地级宗门之一（另两家为**赤焰门**、**碧波轩**），掌控一条中型水木属性灵脉“青岚灵脉”，宗主为元婴中期修士。
*   **特点**：以木、水属性功法见长，擅长培育灵植、炼丹（尤其擅长水系、疗伤类丹药）。门内分为“青木峰”（主战、功法）、“灵溪峰”（主丹、药）两脉，彼此既有合作也有竞争。
*   **现状**：正面临外部压力与内部纷争。外部，与**赤焰门**因一处新发现的灵石矿脉争端不断；内部，宗主寿元将尽，两峰对于下任宗主之位与资源倾斜暗流涌动。

**主角初始境遇**：
*   **身份**：青岚宗**灵溪峰**外门弟子（原为内门天才），因在一次秘境探险中为救同门（实遭陷害）而“道基尽毁”，修为从筑基后期暴跌至炼气一层，且经脉郁结，难以寸进。被贬至宗门下属**云雾坊市**的“百草阁”，担任一名负责处理废丹、药渣的底层执事，受尽冷眼与嘲讽。
*   **环境**：**云雾坊市**位于青岚宗势力范围边境，鱼龙混杂，常有散修、小宗门弟子以及凡人商队来往。百草阁是青岚宗在此处的产业之一，主营丹药收购与售卖，但地位不高。主角居于坊市角落一简陋小院，灵气稀薄。
*   **“修为尽废”的真相（前世记忆揭示）**：并非简单的道基损伤，而是身体被一种极为阴毒的**上古秘术“蚀灵锁脉”**所害，表面看起来是重伤未愈，实则是灵脉被无形锁链禁锢，并不断吞噬生机与灵力。寻常丹药根本无法解除，反而会滋养那道“锁链”。此术罕见，当世无人能识，唯主角凭借仙帝记忆，一眼看穿，并知破解之道始于“至废至虚”之处，需用特殊法门引动残留的仙帝神魂之力，配合数种罕见毒物，以毒攻“锁”，方能破而后立，重铸道基。起点之低微与困境之具体，正为承载仙帝记忆的辉煌归来，铺垫最极致的反差。
