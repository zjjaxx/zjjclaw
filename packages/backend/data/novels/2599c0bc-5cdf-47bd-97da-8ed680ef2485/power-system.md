## 二、 子境界细分与战力公式

*   **四段制**：每个大境界内分为**初期、中期、后期、圆满**。
    *   **初期**：刚突破，力量不稳，需巩固。
    *   **中期**：境界稳固，为常规状态。
    *   **后期**：力量积蓄深厚，开始触及下一境界门槛。
    *   **圆满**：当前境界已达极限，随时可引动突破契机。灵力、神识、肉身均打磨至完美（主角能做到，常人很难）。

*   **战力量化参考**：
    *   **同境小段差距**：约1.5-3倍。例如，一个筑基中期修士，可轻松压制2-3个筑基初期联手。
    *   **大境界壁垒**：**不可逾越的鸿沟**。正常情况下，金丹初期可秒杀任何筑基圆满。这是能量层级和生命层次的本质不同。
    *   **主角越阶逻辑**：
        1.  **道基碾压**：主角的低境界是“完美境界”，灵力质量、神识强度远超同阶，可达到“半步下一境”水平。
        2.  **技术代差**：使用早已失传的上古秘术、禁术，或对法则的初级运用。
        3.  **装备/外物碾压**：用仙帝记忆中的方法，炼制或改造出超规格的法宝、符箓、阵法。
        4.  **战斗艺术**：仙帝级的战斗本能和时机把握，能精准找到对手“道”与“术”中的破绽。

## 三、 金手指与体系的深度挂钩

1.  **认知体系（核心中的核心）**：
    *   **法则认知层级**：当前世界修士认知：**灵气 → 属性之力 → 模糊法则**。

    *   能根据现有资源，自创最适配当前环境的**独家功法**。

3.  **丹、器、阵、符的全知**：
    *   **炼丹**：知晓所有天材地宝的药性本源，可用低级药材通过特殊配比和手法，模拟高级丹药部分药效（“代炼”）。能轻易解除丹毒，改良丹方。
    *   **炼器**：懂得“器纹”与“道纹”的区别，能在低级法器中铭刻高级器纹雏形。
    *   **阵法**：一眼看破阵法核心与生门。能用几块普通灵石，布下让高阶修士头疼的“上古残阵简化版”。
    *   **符箓**：可绘制蕴含一丝“法则真意”的符箓，威力远超同级。

4.  **秘境与机缘的“攻略本”**：
    *   清楚记得各大秘境、遗迹的陷阱、关卡、隐藏奖励的获取方法。
    *   知道哪些“废料”、“古迹”中蕴藏真正的宝物。

5.  **心魔与瓶颈无效化**：
    *   心魔劫？前世已斩三尸，明心见性。
    *   修炼瓶颈？前世走过这条路，所有岔路和坑都记得，知道最平坦的高速公路怎么走。

## 五、 禁忌事项与风险

1.  **走火入魔**：
    *   **成因**：急于求成，灵力失控；心境有缺，被心魔所趁；功法冲突。
    *   **后果**：经脉尽断，修为尽废，或神识错乱沦为只知杀戮的怪物。
    *   **主角免疫**：心境圆满，功法理解通透，几乎不可能走火入魔。

2.  **心魔劫**：
    *   修士内心恐惧、执念、欲望的显化。在突破化神、炼虚、合体、大乘等关键境界时必然出现。
    *   主角视为“补品”或“过场动画”。

3.  **功法冲突**：
    *   同时修炼属性相克或运转路线冲突的功法，会导致灵力在体内争斗，损伤道基。
    *   主角通晓万法，可进行“功法融合”或自创无冲突的复合型功法。

4.  **因果业力**：
    *   肆意屠戮凡人或无度破坏天地，会累积业力。业力深重者，天劫威力会剧增，且心魔劫几乎必死。
    *   主角深知此理，行事会有底线（除非触及逆鳞），且懂得化解业力的秘法。

5.  **夺舍与神魂损伤**：
    *   元婴期及以上修士，元婴/元神离体后可夺舍他人。但成功率低，且有违天道，后患无穷。
    *   主角神魂本质是仙帝级，坚不可摧，只有他夺舍别人（且不屑为之）的份，无人能夺舍他。

6.  **上古秘术反噬**：
    *   如“蚀灵锁脉”这类阴毒秘术，施展者和受害者都可能遭受未知反噬。
    *   主角的知识库中包含这些秘术的**完整版、安全版及反制措施**。
