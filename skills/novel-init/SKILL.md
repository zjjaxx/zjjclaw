# 小说项目初始化

## 职责

收集并整理都市异能网文的核心设定信息，为后续所有创作步骤奠定基础。

## 必须收集的信息

| 字段 | 说明 | 示例 |
|------|------|------|
| title | 小说标题 | 《开局签到系统，震惊全球》 |
| setting | 背景城市 | 华都市、天南市（用虚构城市名） |
| protagonist | 主角姓名及初始身份 | 林凡，普通高中生 |
| cheatType | 金手指类型 | 系统/传承/神器/觉醒/穿越 |
| targetChapters | 目标章节数 | 500-1000 |

## 都市异能金手指类型参考

- **系统流**：游戏化界面，签到/任务/商城/属性面板
- **传承流**：获得强者记忆/功法/传承空间
- **神器流**：得到上古神器、戒指空间、神兵
- **觉醒流**：天赋觉醒，异能天生强大
- **穿越流**：重生/穿越带记忆优势

## 主角初始处境（选一种）

- 被退婚/被嘲笑（经典装逼起点）
- 家族落魄，曾经辉煌
- 普通学生，默默无闻
- 刚被开除/失业（社会底层逆袭）

## 输出格式

确认收集到信息后，返回结构化 JSON：

```json
{
  "title": "小说标题",
  "setting": "背景城市",
  "protagonist": "主角名",
  "cheatType": "金手指类型",
  "targetChapters": 500
}
```
