# 都市异能故事大纲设计

## 网文大纲核心原则

### 爽点密度公式
```
每3-5章：一个小打脸/小突破/小爽点
每15-20章：一个中型高潮（大打脸/境界突破/获得资源）
每50章：一个阶段性大高潮（弧线终结/势力洗牌）
```

### 主角成长三要素（缺一不可）
1. **力量成长**：境界突破节奏（不能太慢，要让读者爽）
2. **地位成长**：从底层到被人仰望
3. **格局成长**：从个人恩怨到守护更大的存在

## 动态弧线划分

根据 `targetChapters` 按比例计算各弧章节范围，严格按照以下比例切割：

| 总章节数 | 弧线数 | 各弧占比（依次） |
|----------|--------|-----------------|
| ≤ 100    | 2弧    | 40% / 60% |
| 101–200  | 3弧    | 20% / 35% / 45% |
| 201–350  | 4弧    | 15% / 25% / 35% / 25% |
| 351+     | 5弧    | 10% / 20% / 30% / 25% / 15% |

**计算规则**：章节范围 = 上一弧结束章 + 1 到 round(targetChapters × 累计占比)

用户 prompt 中会提供已计算好的弧线章节范围建议，直接采用即可。

## 各弧核心任务

### 第一弧：觉醒与初现锋芒

**必须包含**：
- 开篇被欺负场景（让读者恨反派）
- 金手指触发（第1-3章内，不能拖）
- 第一次装逼打脸（本弧前20%内）
- 家族/亲人危机（增加紧迫感）
- 第一个小境界突破

### 第二弧：崭露头角

**核心任务**：
- 在城市中建立名声
- 打脸更强的对手
- 获得第一批真正的资源/人脉
- 感情线初步展开

### 第三弧：势力角逐

**核心任务**：
- 创立自己的势力或掌握某个组织
- 对抗城市级反派
- 中级BOSS登场并初次交手
- 主角开始影响城市格局

### 后续弧

每弧格局扩大一级：城市 → 省级 → 全国 → 更大

## 打脸节奏设计

### 打脸的层次递进

| 阶段 | 打脸对象 | 打脸方式 | 观众反应 |
|------|----------|----------|----------|
| 早期 | 同学/前任 | 展示实力/财富 | 惊愕 |
| 中期 | 家族对立者 | 当众羞辱 | 跪地求饶 |
| 后期 | 组织大佬 | 实力碾压 | 众目睽睽下折腰 |

**打脸必要元素**：
1. 对方先嚣张（越嚣张越好）
2. 主角沉默不解释
3. 关键时刻反转
4. 对方当众出丑
5. 围观者震惊/议论

## 感情线节奏（按占比换算）

- **前5%章节**：铺垫，女主出场但关系普通
- **5%–20%**：互动增多，女主对主角改观
- **20%–40%**：感情线升温，出现情敌/危机
- **40%以后**：确定关系，但保持一定张力
