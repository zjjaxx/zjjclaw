# 都市异能故事大纲设计

## 网文大纲核心原则

### 爽点密度公式
```
每3-5章：一个小打脸/小突破/小爽点
每15-20章：一个中型高潮（大打脸/境界突破/获得资源）
每50章：一个阶段性大高潮（弧线终结/势力洗牌）
```

### 主角成长三要素（缺一不可）
1. **力量成长**：境界突破节奏（不能太慢，要让读者爽）
2. **地位成长**：从底层到被人仰望
3. **格局成长**：从个人恩怨到守护更大的存在

## 故事弧设计

### 第一弧：觉醒与初现锋芒（1-50章）

**核心任务**：
- 主角在最低谷获得金手指
- 第一次让周围人刮目相看
- 打脸最初的欺负者
- 完成第一个小境界突破

**必须包含**：
- 开篇被欺负场景（让读者恨反派）
- 金手指触发（第1-3章内，不能拖）
- 第一次装逼打脸（第5-10章内）
- 家族/亲人危机（增加紧迫感）

### 第二弧：崭露头角（51-150章）

**核心任务**：
- 在城市中建立名声
- 打脸更强的对手
- 获得第一批真正的资源/人脉
- 感情线初步展开

### 第三弧：势力角逐（151-300章）

**核心任务**：
- 创立自己的势力或掌握某个组织
- 对抗城市级反派
- 中级BOSS登场并初次交手
- 主角开始影响城市格局

### 后续弧（300章+）

每弧格局扩大一级：城市→省级→全国→更大

## 打脸节奏设计

### 打脸的层次递进

| 阶段 | 打脸对象 | 打脸方式 | 观众反应 |
|------|----------|----------|----------|
| 早期 | 同学/前任 | 展示实力/财富 | 惊愕 |
| 中期 | 家族对立者 | 当众羞辱 | 跪地求饶 |
| 后期 | 组织大佬 | 实力碾压 | 众目睽睽下折腰 |

**打脸必要元素**：
1. 对方先嚣张（越嚣张越好）
2. 主角沉默不解释
3. 关键时刻反转
4. 对方当众出丑
5. 围观者震惊/议论

## 感情线节奏

- **第1-30章**：铺垫，女主出场但关系普通
- **第30-100章**：互动增多，女主对主角改观
- **第100-200章**：感情线升温，出现情敌/危机
- **200章+**：确定关系，但保持一定张力
