# 网文章节审校

## 审校维度

### 1. 爽感检查 ✓/✗

**快速自测**：
- 这章有打脸/突破/装逼吗？
- 读者会"爽"吗？如果没有，是否有足够的"蓄力"？
- 3章内有没有一个爽点？

**常见问题**：
- 章节全是铺垫，没有任何爽感 → 加入小打脸或伏笔揭晓
- 爽点太快，没有铺垫 → 反派不够嚣张，读者没有期待

### 2. 结尾钩子检查

**合格钩子标准**：
- 最后一句话让人想知道下一章发生什么
- 不能以"主角睡了/主角想了很多/这一天结束了"收尾

**检查清单**：
- [ ] 最后一段有强烈停顿感？
- [ ] 最后一句有冲击力？
- [ ] 钩子类型是否合适（悬念/反转/爽感/危机）？

### 3. 网文语感检查

**语言问题清单**：
- [ ] 有无过长段落（超过6行）？→ 拆分
- [ ] 有无过于书面化的表达？→ 改口语/动作化
- [ ] 对话是否自然？有无说教感？
- [ ] 有无大量景物/心理描写影响节奏？

**常见问题句型**：
- "此刻的林凡，心中涌起了一股莫名的感慨……"（太文学，减���）
- 大段环境描写（3行以上，立刻压缩到1行）
- 连续多个"他想到了……"（内心戏过多）

### 4. 逻辑一致性检查

**战力一致性**：
- 本章主角的战力是否与前文一致？
- 反派的实力是否合理？
- 有无"主角突然变弱"的剧情需要？（需要有充分理由）

**情节一致性**：
- 人物的行为是否符合其性格设定？
- 场景切换是否合理？
- 时间线有无矛盾？

### 5. 节奏检查

**节奏是否平衡**：
- 动作场景：打斗节奏要快，段落要短
- 对话场景：不要连续说超过5句没有动作描写
- 过渡段落：不要太长，2-3句带过即可
