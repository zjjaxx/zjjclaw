# 打脸情节设计

## 打脸的本质

打脸 = **期待感的建立** + **反转时刻** + **情绪释放**

读者读打脸章节的心理路径：
```
"这个坏人太可恨了" → "主角快出手" → "终于！" → "太爽了！"
```

## 打脸的五个阶段

### 第一阶段���铺垫（让读者恨）
- 反派必须先嚣张，而且要嚣张到位
- 必须有观众（围观群众=读者代入）
- 主角此时处于劣势/沉默

**铺垫台词模板**：
> "就凭你？哈哈哈，你知道我是谁吗？"
> "废物就是废物，这辈子都翻不了身。"
> "我劝你跪下来道歉，我还可以考虑放你一马。"

### 第二阶段：主角的沉默（蓄力）
- 主角**不要急着反击**，让反派嚣张够
- 主角的沉默本身就是压迫感
- 偶尔一句话，让反派警惕但没放在心上

**沉默时的主角反应**：
> 林凡没有说话。
> 他只是看着对方，眼神很平静。
> 反而是这种平静，让陈浩南隐隐不安。

### 第三阶段：反转触发器
- 某个证明/展示触发反转
- 可以是：实力展示/身份披露/资源展示/第三��证明

**触发器类型**：
- 电话（"喂，是我，把那件事办了。"）
- 证件/文件（扔出一个东西，对方看完脸色大变）
- 直接动手（一击制敌）
- 第三方出现（"这位是……天！"）

### 第四阶段：反转执行（爽点核心）
- 速度要快，不要拖泥带水
- 让反派当众出丑
- 围观者的反应是爽感的放大器

**围观者反应层次**：
```
懵逼 → 惊讶 → 倒抽凉气 → 交头接耳 → 偷眼看反派
```

**反派被打脸时的反应**：
- 第一反应：不敢相信（愣住）
- 第二反应：慌乱（退后一步/脸色发白）
- 第三反应：尝试挽回面子（说些无力的话）
- 第四反应：彻底崩溃（���抖/语无伦次/跪地）

### 第五阶段：主角收尾（余味）
- 主角不要多话
- 一句话或不说话，转身离开
- 留下一个"背影"更有意境

**收尾台词**：
> "这就是全部吗？"
> "好好保重。"（根本没把对方放在眼里）
> "我们的事，到此结束。"
> （什么也不说，转身就走）

## 打脸场景的围观设计

好的打脸需要足够多的"观众"：
- 选择公���场合：食堂/宴会/公司大厅/校园广场
- 描写观众的反应（但不要太多，10-20字就够）
- 重要人物在场（女主目睹会加分）

## 特殊打脸类型

### 延迟打脸（更爽）
反派嚣张 → 主角默默记下 → 数章后，在反派最春风得意时反转

### 连环打脸
打完小弟打老板，打完老板打老板的老板，层层递进

### 当着最在乎的人打脸
在反派的未婚妻/父母/上司面前打脸，羞辱感×10

## 输出

设计打脸情节时，提供：
1. 场景设置（地点+在场人物）
2. 铺垫台词（反派的嚣张）
3. 主角的反应过程
4. 反转触发器
5. 反转执行（要有画面感）
6. 围观者反应
7. 收尾台词
