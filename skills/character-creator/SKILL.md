# 都市异能人物创建

## 人物体系结构

### 1. 主角（protagonist）

都市异能主角标配：
- **外表**：普通甚至寒酸（反差爽感）
- **性格**：表面淡定/内心热血，关键时刻霸气
- **初始境遇**：必须有"低谷"——被嘲笑、被退婚、被欺负
- **核心驱动**：保护家人/复仇/证明自己
- **特殊禁忌**：不能太圣母，不能太傻白甜

### 2. 女主/感情线角色（female_lead）

配置 2-3 个女性角色，形成竞争关系：
- **青梅竹马型**：从小认识，暗恋主角，忠诚度高
- **傲娇类型**：初期看不起主角，被打脸后爱上
- **神秘背景型**：来历不凡，和主角有隐秘联系

每个女主需要：明确的外貌特征 + 鲜明性格 + 与主角的感情进展节拍

### 3. 反派梯队（分三级）

**小反派（cannon_fodder）**：
- 每弧初期的嚣张炮灰，负责被打脸
- 通常是主角身边的"贱人"：前任/同学/同事
- 嚣张程度要强，这样打脸才爽
- 台词示例："就你这种废物也配……"

**中级反派（antagonist）**：
- 家族对立者/组织敌人，有一定实力
- 多次对主角形成威胁，有来有往
- 必须有自己的目标和逻辑，不能纯粹为坏而坏

**大Boss（villain_boss）**：
- 幕后黑手，前几十章不直接出场
- 与主角有深层矛盾（杀父之仇/夺宝之恨）
- 实力要远超主角现有境界

### 4. 配角支持团（supporting）

- **铁哥们/师父型**：帮主角，偶尔搞笑
- **神秘老者**：看出主角天赋，暗中扶持
- **家族长辈**：需要主角保护，提供情感驱动

## 人物性格模板禁忌

❌ 不要创建：所有人都支持主角、没有真正威胁的反派、毫无立场的工具人
✅ 需要确保：反派要让读者恨，女主要让读者喜欢，哥们要让读者觉得真实

## 输出格式

输出 JSON 数组，每个人物包含：

```typescript
{
  name: string,
  role: "protagonist" | "female_lead" | "antagonist" | "villain_boss" | "supporting" | "cannon_fodder",
  age: number,
  appearance: string,      // 外貌描述（2-3句）
  personality: string,     // 性格特点（2-3句）
  background: string,      // 背景故事（3-4句）
  abilities: string,       // 已有能力/潜力
  currentRealm: string,    // 当前境界
  relationshipToProtagonist: string  // 与主角的关系
}
```

人物数量：主角1 + 女主2-3 + 小反派2-3 + 中反派1-2 + 大Boss1 + 配角2-3
